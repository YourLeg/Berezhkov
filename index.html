Подготовка к экзамену

Один большой вопрос к экзамену, который можно разделить на несколько маленьких, в случае, если мой вопрос случайно пересечется с чьим-нибудь еще.

Вопрос про ip-адресацию. Предположим, есть компьютер A с ip-адресом 192.168.128.128 в сети с маской 255.255.0.0.
    Есть адреса:
  1. 192.168.0.0
  2. 192.168.1.0
  3. 192.168.0.1
  4. 192.168.128.128
  5. 192.168.255.0
  6. 192.168.0.255
  7. 192.168.255.255
  8. 127.0.0.1
    Ответьте на вопросы:
  1. Какие из перечисленных адресов лежат в той же сети?
  2. Какие из перечисленных адресов могут быть у узлов в данной сети?
  3. Какие из перечисленных адресов могут быть адресом назначения в данной сети, если пакет шлет компьютер А?
  4. Какие адреса может пинговать компьютер А с уверенностью, что пинг завершится удачно?(если нет уверенности, что в сети вообще есть другие узлы)
  5. Какие адреса может пинговать компьютер А с уверенностью, что пинг завершится удачно?(в сети есть хотя бы один другой узел)
    Ответы:
  1. 1, 2, 3, 4, 5, 6, 7
  2. 2, 3, 4, 5, 6
  3. 1, 2, 3, 4, 5, 6, 7
  4. 1, 4, 7, 8
  5. 1, 4, 7, 8

Ответ на данный вопрос можно получить практически с помощью Cisco Packet Tracer, либо изучив теорию про сетевые адреса, широковещательный адрес, особые IP-адреса и локальный хост
